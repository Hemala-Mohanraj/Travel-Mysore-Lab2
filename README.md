# Travel Mysore

## Project Description

Travel Mysore is a responsive tourism website developed using HTML and CSS. The website provides information about the city of Mysore, including its popular tourist attractions, hotels, travel information, image gallery, and contact details. The project is designed to offer visitors a simple and visually appealing platform to explore Mysore.

## Live Website

https://hemala-mohanraj.github.io/Travel-Mysore-Lab2/index.html

## Features

- Responsive user interface
- Sticky navigation bar
- Hero section with background image
- About Mysore section
- Tourist attractions displayed using cards
- Hotel recommendations
- Image gallery
- Statistics section
- Testimonials section
- Travel information table
- Frequently Asked Questions (FAQ)
- Contact form
- Footer with social media links

## How to Run the Project

1. Clone the repository.

```bash

```

2. Open the project folder.

3. Open `index.html` in any modern web browser.

No additional software or dependencies are required.
